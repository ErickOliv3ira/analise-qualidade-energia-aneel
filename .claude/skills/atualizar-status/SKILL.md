description: Atualiza docs/status_progresso.md ao fim de uma sessão de trabalho, marcando itens concluídos e definindo a próxima ação. Usar quando o usuário disser que terminou uma tarefa, fase ou sessão.
---

Revise o que foi feito nesta sessão e atualize `docs/status_progresso.md`:

1. Marque com [x] os itens efetivamente concluídos (só marque o que foi testado/verificado, não o que foi apenas escrito)
2. Atualize os campos "Fase atual", "Última atualização" (data de hoje) e "Próxima ação" (a próxima tarefa concreta do roadmap)
3. Se surgiu bloqueio ou pendência, adicione na seção "Bloqueios / pendências atuais"
4. Se surgiu ideia fora do escopo, adicione em "Ideias estacionadas" — NUNCA implemente
5. Mostre um diff resumido do que mudou no arquivo
6. Lembre o usuário de subir a versão atualizada também no conhecimento do Claude Project (arquivo 05)
